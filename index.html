多端发布

是一个使用 Vue.js 开发所有前端应用的框架，开发者编写一套代码可发布到多个平台

# 就像数 1, 2, 3 一样容易


  # 安装
  npm install uni-simple-router #  或者： yarn add uni-simple-router

  #  开始开发
  npm run dev

  #  构建静态文件
  npm run build
